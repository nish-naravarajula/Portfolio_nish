## Instructions to Build

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- [Git](https://git-scm.com/) installed

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/nish-naravarajula/portfolio.git
```

2. **Navigate to the project**
```bash
   cd portfolio
```

3. **Install dependencies**
```bash
   npm install
```

4. **Run ESLint** *(optional)*
```bash
   npx eslint js/main.js js/terminal.js
```

5. **Open in browser**
   - Open `index.html` directly in your browser
   - Or use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
